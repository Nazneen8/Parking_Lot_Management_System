# PARKING LOT MANAGEMENT SYSTEM

A web-based **Parking Lot Management System** designed to streamline parking operations for students, staff, and administrators. 
This system provides secure login, real-time parking bay availability, payment processing, and administrative management â€” all in one platform.

---
The Parking Lot Management System is a digital solution developed for **Rosebank College** to manage parking efficiently.  
It offers an intuitive web interface where users can:
- Log in using their role (Student or Staff)
- View and reserve parking bays
- Process payments (for students)
- Receive notifications for payment or expiry reminders
- Manage system data via an admin dashboard

# FEATURES

# Student
- Secure login using Rosebank student email
- View available parking slots
- Choose and reserve a parking bay
- Select a payment plan
- Process and verify payments
- Receive expiry or renewal notifications

# Staff
- Log in with staff credentials
- Access free parking allocation
- View parking bay availability
- Manage basic parking operations (no payment required)

# Admin
- Secure admin login
- Monitor parking bay usage across all users
- View student payments and new signups
- Manage users, parking bays, and notifications
- View summary of parking allocations (Students / Staff / Guests)
